一、积累与运用
第1题：根据拼音写汉字
分值：4分

根据拼音在田字格内写出相应的汉字。

①辨伪去wàng ( ) ②孜孜不倦jí取 ( ) ③不能一味mò守 ( ) ④自bào自弃 ( )

参考答案

妄、汲、墨、暴

解题思路

1. "辨伪去wàng"：根据上下文"怀疑不仅是消极方面辨伪去wàng的必须步骤"，"辨伪去wàng"是指辨别虚假、去除虚妄，所以是"妄"。

2. "孜孜不倦jí取"：意思是勤奋努力不知疲倦地获取知识，"汲"有吸取、获取的意思。

3. "不能一味mò守"：指不能一味地拘泥、遵守旧的说法，"墨守"指固执地遵守旧规。

4. "自bào自弃"：指自甘堕落，不求进取，"自暴自弃"是固定成语。

第2题：标点符号选择
分值：2分

填入【甲】【乙】两处的标点，最恰当的一项是（ ）

A. 【甲】"。 【乙】； B. 【甲】"。 【乙】，

C. 【甲】。" 【乙】， D. 【甲】。" 【乙】；

参考答案

D

解题思路

【甲】处：前文是直接引语"我太无能，不能创造"，后面是说话人的叙述，所以句号应放在引号里面。

【乙】处："鲁钝的曾参，传了孔子的道统"和"不识字的慧能，传了黄梅的教义"是两个并列分句，中间应该用分号。

因此正确答案是D。

第3题：语法知识判断
分值：2分

下列说法正确一项是（ ）

A. "学贵有疑""基本条件""jí取知识""自bào自弃"短语类型各不相同。

B. 文段中画波浪线的句子是并列复句。

C. 文段画线句的主干是：我们mò守旧说。

D. "鲁钝"是"愚笨，不敏锐"的意思，这里是谦辞。

参考答案

A

解题思路

分析各选项：

B选项：画波浪线的句子"怀疑不仅是消极方面辨伪去妄的必须步骤，也是积极方面建设新学说、启迪新发明的基本条件"是递进复句，不是并列复句。

C选项：画线句"我们不能一味mò守前人的旧说"的主干是"我们不能mò守旧说"，原选项缺少"能"。

D选项："鲁钝"是形容曾参的，这里是客观描述，不是谦辞。

A选项正确："学贵有疑"是主谓短语，"基本条件"是偏正短语，"jí取知识"是动宾短语，"自bào自弃"是并列短语，四者类型确实各不相同。

第10题："僵卧孤村不自哀，尚思为国戍轮台。夜阑卧听风吹雨，铁马兵河入梦来。"------《十一月四日风雨大作》，你认为这首诗歌放在陆游的哪一段起落中最合适，请说明理由。

__________

参考答案

第6题：①批评，指责 ②修建，整理，治理 ③A ④适意，舒适

第7题：但 见 陆 子 之 容/ 夷 然 澹 然/ 莫 不 叹 以 为 不 可 及 也

第8题：示例一：《书愤》中"塞上长城空自许，镜中衰鬓已先斑。"中"空"有"徒劳地""白白地"之意，形象地写出了因宏伟志向无法实现而产生的沉重的失落感，"鬓已先斑"意思是头发已白，实际上揭示了自己屡受排挤、理想落空的处境，表达了诗人壮志难酬的悲愤之情。既有壮志未酬的悲愤又有对现实无奈的感慨。

示例二：《诉衷情·当年万里觅封侯》中"胡未灭，鬓先秋，泪空流。"说尽平生不得志。回首人生，流年暗度，两鬓已苍；沉思往事，雄心虽在，壮志难酬。"未""先""空"三字在承接比照中，流露出内心失望沉痛、报国无门的情感。

第9题：①陆游因与范成大以文字结交，不拘泥于礼节，被人嘲笑颓放，因此自号"放翁"以表达自己对世俗眼光的不屑，体现了陆游豪放不羁、超凡脱俗的个性；②陆游一生充满了对自然与生活的热爱，他热爱山水，不被世俗所累，与自然相融，追求心灵的宁静与自由，这在他的诗作《游山西村》中有所体现；③"放"也有放下、释放之意。陆游一生历经坎坷，多次遭贬谪，但他始终保持着乐观豁达的心态。

第10题：放在四起四落中。理由："铁马兵河入梦来"，诗人借助梦境表达自己虽年迈，但仍有报国的壮志雄心。空怀壮志却无法被重用，只能"僵卧孤村"。与第四起四落中的《诉衷情》中的"此生难料，心在天山，身老沧洲。"内容情感相似。

文言文翻译

《宋史·列传·卷一百五十四》译文：范成大统帅蜀地（担任四川制置使）时，陆游担任参议官，（二人）凭借文字结交，不被礼节和法度所拘束，人们讥讽陆游行为颓废放旷，于是他就自号为"放翁"。后来陆游多次升迁，担任了江西常平提举（负责管理常平仓救济、农田水利等事务的官职）。

《东篱记》译文：陆游家房子的东边一侧，有一个小园子，距离喧闹的集市并不远，然而园子里草木的意趣，与山林间的并没有什么不同。我（陆游，自号陆子）修整了一间小屋住在里面，给它取名叫"东篱"，是取用了陶渊明"采菊东篱下"这句诗的意思。东篱的南边，有几丛菊花，开花的时候色彩绚丽，香气充满了整个屋子。南边的窗户用竹子做成窗格，阳光和月色照进来，都非常惹人喜爱。东篱的西边，有几竿竹子，粗得像椽子一样，长长的枝叶低垂，几乎拂到地面。风吹过来时，竹子发出萧萧的声响，下雨的时候，雨水打在竹叶上淅淅沥沥，这些都成了美妙的声音。我每天住在这间屋子里，有时带着书来阅读，有时提笔写作，有时采摘菊花来欣赏把玩，有时除草来栽种花草，有时煮茶来品尝，这些都让我感到很舒适惬意。别人来到我这里，也不再问其他的事情，只看到我神态平和、淡泊的样子，没有不感叹认为这是常人难以企及的境界。

解题思路

第6题：文言词语解释要掌握方法：联系课内法、结合语境法、查阅字典法、组词法等。

第7题：断句要理解句意，注意主语和谓语之间、并列成分之间可以适当断开。"夷然澹然"是并列短语，形容陆游的神态，前后可断。

第8题：诗歌赏析要抓住关键词，分析其含义和表达效果，体会诗人情感。"空""已""未""先"等字都是理解诗歌情感的关键。

第9题：理解"放翁"含义要结合陆游的生平经历和诗文作品，从个性、生活态度和心境三个方面分析。

第10题：诗歌归类要抓住诗歌内容和情感，与陆游不同阶段的人生经历和心境进行匹配。

解题思路

第11题：理解名著中的人物形象，要结合具体情节和人物性格特点。

  • 晁盖：《水浒传》中梁山泊的第二任首领，仗义疏财，人称"托塔天王"。
  • 沈琼枝：《儒林外史》中具有反抗精神的女性形象，自尊自强。

第12题：名著知识辨析题要准确掌握作品中的人物形象和情节。马二先生是《儒林外史》中热衷科举的腐儒，不是贪官污吏。
